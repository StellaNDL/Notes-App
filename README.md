# Notes App

A simple fullstack Notes application built with **React**, **Express**, **Supabase**, and **Node.js**.  
Users can register, login, and manage their private notes with full **CRUD** functionality.

---

## Features

- User authentication with Supabase
- Add, edit, delete, and view notes
- Notes are private per user
- Frontend built with React + Tailwind CSS
- Backend built with Express and Supabase
- Fully deployable to Vercel
